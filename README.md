how to play
-----------
- keys >
	- `w` => `up`
	- `s` => `down`
	- `d` => `right`
	- `a` => `left`

- خبیس کیست؟
	- موجود بانمکی که نباید گول ظاهرش را بخورید
	- این موجود با نمک دزد میوه است و شما باید جلوی اورا بگیرید
